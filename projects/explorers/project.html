Explorers the Game

The purpose of “Explorers the Game” is to create a fun way for children in elementary schools to learn about geography, world exploration history, and stories about famous historical explorers and figures using bluetooth beacon technology

The Design:

The project is a phone application where the user can choose to explore with one of the explorers, and follow the prompt direction to walk around an area and discover new lands, just like those explorers did back then. A total of four beacons are used, representing Asia, Europe, Africa and America. When the user gets close to the beacon, the phone app will indicate that the user has arrived in the corresponding continent, and tell the user a little bit about what the explorers did at that place. At each location the user has the option to switch explorers, and choose to explore as another explore, and start an entirely different exploration route.

Technologies Used:

  1. Node.js for the app
  2. Apache Cordova for translation to mobile
  3. iBeacon and bluetooth beacons for interactions
  4. Evothings Studio for bluetooth integration
  5. html, css, & javascript for layout
  6. photoshop for graphics assets

This project will not only help the children to learn about our world, but also encourage them to engage in some level of physical activities.
