TrustWorth

TrustWorth is a multi-sequential web based narrative akin to a comic book.

View the project here

The Design:

The narrative is meant to encourage exploration within the story and to encourage replay for different outcomes. The user is expected to utilize social and contextual clues to advance the story.

Project Needs:

  1. a system of trust
  2. a moral dilemma
  3. multiple distinct outcomes
  4. multiple objectives

Technology Used:

  1. HTML
  2. CSS
  3. JavaScript
